# QAInfomax  
*An implementation of the [QAInfomax: Learning Robust Question Answering System by Mutual Information Maximization](https://arxiv.org/abs/1909.00215)*

# Requirements

python 3.7

```
pip install -r requirement.txt
```

# Train:

```
sh train.sh
```

# Evaluation:

```
cd squad_adv
sh eval_adv.sh
```
